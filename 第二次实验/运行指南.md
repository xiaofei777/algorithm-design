- 共有 6 个加油站

- 需要在第 3、5、6 号加油站加油，共加油 3 次

## 六、输入文件格式说明

input.txt 文件格式：

```
n k
d1 d2 d3 ... dk+1
```

| 参数 | 说明 |
|------|------|
| n | 汽车加满油可行驶的最大距离（整数） |
| k | 加油站的数量（整数） |
| d1~dk+1 | k+1 个整数，表示各段距离 |

**加油站编号说明：**
- 第 0 号：出发地（起点）
- 第 1~k 号：沿途的加油站
- 第 k+1 号：目的地

## 八、常见问题

### Q1: 编译时提示 "command not found: gcc"

**A:** 需要先安装 gcc 编译器。
- macOS: `xcode-select --install`
- Ubuntu/Debian: `sudo apt-get install gcc`
- Windows: 安装 MinGW 或 Visual Studio

### Q2: 运行时提示 "无法打开 input.txt 文件"

**A:** 确保当前工作目录正确，或者 input.txt 文件存在于当前目录中。

### Q3: 输出 "No Solution"

**A:** 说明存在某两个加油站之间的距离超过了汽车加满油可行驶的最大距离，无法到达目的地。

### Q4: 如何截图实验结果？

**A:** 
- macOS: 按 `Command + Shift + 4` 选择区域截图
- Windows: 按 `Win + Shift + S` 或使用截图工具
- Linux: 使用系统自带的截图工具
